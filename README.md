# finance-paperdaily

Daily financial reports · HTML only · deployed to <https://finance.paperdaily.org>

- `index.html` — latest day (auto-refreshed each publish)
- `YYYY/MM/DD.html` — archive by date
- `history.html` — browse all past issues

Content is machine-generated by the finance-daily-report skill and published via
`skills/finance-daily-report/tools/publish_public.sh`. Do not hand-edit.
